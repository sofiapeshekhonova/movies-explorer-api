## Описание проекта

Библиотека фильмов, с возможностью добавить фильмы в избранное и затем удалить.
 
**Технологии:** Eslint, Node.js, Express, MongoDB

## Запуск проекта

`npm run start` — запускает сервер   
`npm run dev` — запускает сервер с hot-reload
